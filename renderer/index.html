Komponenten

Wähle aus, was in dein SD-Pack kommt – Versionen kommen immer direkt von GitHub.

GitHub-Token optional

GitHub erlaubt ohne Anmeldung nur 60 Versions-Abfragen pro Stunde – danach erscheint „Rate-Limit erreicht“. Mit einem kostenlosen Token (keine Berechtigungen nötig) sind es 5.000. Der Token wird nur lokal auf diesem PC gespeichert.
